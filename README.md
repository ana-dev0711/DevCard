# DevCard

Aplicativo mobile desenvolvido com React Native e Expo Router.

## Funcionalidades

1- Tela de boas-vindas
2- Cadastro com validação
3- Preview do cartão
4- Tela de sucesso
5- Navegação com Expo Router

## Tecnologias utilizadas

- React Native
- Expo
- TypeScript
- Expo Router
